# Criar QR Code

Ferramenta para geração de QR Codes (SVG e PNG) com alta correção de erro.

## Uso Rápido
```bash
python criar-qrcode/generate_qrcode.py
```
Edite `data_to_encode` e `output_path` no script.

## Estrutura
```
criar-qrcode/
  generate_qrcode.py
  MANUAL.md
  README.md
```

## Dependências
- qrcode
